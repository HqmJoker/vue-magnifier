## 准备
安装vue3相关环境

1. 安装npm环境（ps:可以网上搜索教程）

2. 安装Vue CLI脚手架

   ```bash
   npm install -g @vue/cli
   ```

3. 创建项目并运行

   ```bash
   npm init vite <project-name> -- --template vue
   cd <project-name>
   npm install
   npm run dev
   ```
